## 1. Difference between var, let, and const
var: Function scoped, can be redeclared and updated.  
let: Block scoped, can be updated but not redeclared.  
const: Block scoped, cannot be updated or redeclared.

## 2. What is the spread operator (...)?
The spread operator (...) is used to expand elements of an array or object.

## 3. Difference between map(), filter(), and forEach()
map(): Returns a new array after modifying each element.  
filter(): Returns a new array based on a condition.  
forEach(): Loops through an array but does not return a new array.

## 4. What is an arrow function?
An arrow function is a shorter way to write functions in JavaScript using =>.

## 5. What are template literals?
Template literals use backticks (` `) to create strings and insert variables easily.
